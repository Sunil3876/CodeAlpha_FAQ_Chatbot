# NLP FAQ Chatbot 🤖

## 📌 Overview
Developed during the CodeAlpha Artificial Intelligence Internship. This intelligent chatbot answers frequently asked questions by finding the best semantic match to the input.

## 🧠 Core Technologies
* **Tech Stack:** Python, NLTK, Scikit-Learn.
* **Algorithms:** TF-IDF Vectorization and Cosine Similarity.

## 🛠️ How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. Run the chatbot: `python chatbot.py`
